        -> Page HTML and all static assets (CSS, fonts) are loaded

Step 3: Verify the username input box is present on the page

Step 4: Verify the registration button label shows "לינק להרשמה ➔"

Step 5: Type "Moshe Mamia" into the username input field

Step 6: Click the "לינק להרשמה ➔" button
        -> JavaScript alert is triggered (POST attempt captured in HAR)

The HAR file captures all HTTP requests and responses during this scenario,

It can be used to analyze network performance, response times, and payload sizes.
